# Changelog

All notable changes to CachePhoenix will be documented in this file.

## [1.0.0] - 2026-02-27

### Added
- Initial release as CachePhoenix
- Auto-detect browser cache folders for Discord, Chrome, Brave, Edge, and Opera
- Chunked MP4 video reconstruction from Chromium cache chunks
- Support for MP4, WebM, PNG, JPEG, GIF, WebP, and AVI recovery
- Live thumbnail preview generation during scanning
- WebM/GIF to MP4 conversion option
- Organize recovered files by type into subfolders
- Custom output folder selection
- Scan state preservation across page navigation
- Sort and filter scan results by type and date
- Cross-platform support: Windows, macOS (ARM + Intel), Linux
- Bundled ffmpeg/ffprobe sidecars (no external dependencies)
